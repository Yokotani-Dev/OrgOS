### 2. `scripts/org/acquire-lease.sh` (新規)

```bash
acquire-lease.sh \
  --task-id T-XXX \
  --actor-role codex|manager|subagent \
  --actor-id <id> \
  --allowed-paths "path1,path2,path3" \
  [--worktree-path <path>] \
  [--branch <name>] \
  [--ttl-seconds 1800]  # default 30 min
```

#### 動作
1. 入力検証 (task_id, actor, allowed_paths 必須)
2. **conflict check**: 既存 active lease を全 scan し、allowed_paths overlap 検出 (path_covers 同じロジック)。
3. overlap あれば exit 3、conflict 詳細を stderr に。
4. lease_id 生成: `LS-<ts>-<task_id>-<8hex>`
5. lease JSON 構築 (schema v1, status=active, expires_at=now+ttl)
6. `.ai/leases/<lease_id>.json` に atomic write (tmp file + mv)
7. stdout に lease_id

### 3. `scripts/org/release-lease.sh` (新規)

```bash
release-lease.sh <lease_id> [--reason done|cancelled|expired]
```

#### 動作
1. lease file 存在確認
2. status を `released` に更新、`released_at` 記録
3. file を `.ai/leases/.released/<lease_id>.json` に move (history)

### 4. `scripts/org/list-leases.sh` (新規)

```bash
list-leases.sh [--task-id T-XXX] [--actor-role X] [--include-expired]
```

#### 動作
1. `.ai/leases/*.json` を読む
2. heartbeat_at + ttl で expiry 自動判定、expired は status=expired に in-place 更新 (Garbage collection)
3. active lease を tabular 表示 (lease_id / task_id / actor / paths / age)

### 5. `.claude/hooks/pretool_policy.py` 拡張 — Invariant #4 enforce

```python
def evaluate_invariant_violations(tool, command, path, cwd, env=None):
    violations = []
    # ... 既存 logic (Invariants #1, #2, #3, #5) ...
    
    if tool in {"Edit", "Write"}:
        # State Mutation via Org Tool (#5) — 既存
        # ...
        
        # Lease Before Write (#4) — 新規
        if not is_outside_repo(path) and not is_protected_state_file(path):
            active_leases = load_active_leases(cwd)
            if not any_lease_covers(active_leases, path):
                violations.append(("LeaseBeforeWrite",
                    f"write to {path} without active lease covering it"))
    
    return violations


def load_active_leases(cwd):
    """Read .ai/leases/*.json, return list of {allowed_paths, expires_at} for active."""
    leases_dir = os.path.join(cwd, ".ai/leases")
    if not os.path.isdir(leases_dir):
        return []
    result = []
    for fn in os.listdir(leases_dir):
        if not fn.endswith(".json"):
            continue
        try:
            with open(os.path.join(leases_dir, fn)) as f:
                lease = json.load(f)
                if lease.get("status") == "active" and not is_expired(lease):
                    result.append(lease)
        except Exception:
            continue
    return result


def any_lease_covers(leases, path):
    """Path-prefix match (re-use path_covers logic similar to audit)."""
    for lease in leases:
        for ap in lease.get("allowed_paths", []):
            if path_covers(ap, path):
                return True
    return False
```
