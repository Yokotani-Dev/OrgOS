## Background

T-OS-418 dogfood iter4 で:
```
diff budget exceeded: 2296 files > 53
```

Root cause (確認済み): `scripts/org/request-integration.sh` L194:
```python
allowed_paths = status_paths() or ["**"]
```

これは worktree 内の全 changed paths を queue item の `scope.allowed_paths` にする。lease の allowed_paths (`docs/kernel-v2/`) は無関係に整理されておらず、結果として queue item は 2296 paths (含む .ai/ARTIFACTS/, .ai/sessions/, etc.) を allowed と宣言。

## Fix

`scripts/org/request-integration.sh` に `--allowed-paths "p1,p2"` CLI 引数を追加。優先順:

1. `--allowed-paths` 明示指定 → それを使う
2. 不在 + active lease (`.ai/leases/<lease_id>.json` で task_id 一致 + status=active) あり → lease の allowed_paths を使う
3. 両方不在 → exit 2 + error message ("allowed_paths required: --allowed-paths or active lease")

旧 fallback "status_paths()" は廃止 (これが bug の元)。

## Required Implementation

### 1. CLI 引数追加
```bash
--allowed-paths "p1,p2,..."   # comma-separated
```

### 2. lease lookup

```python
def find_active_lease_for_task(task_id):
    leases_dir = Path(".ai/leases")
    if not leases_dir.is_dir():
        return None
    for f in leases_dir.glob("*.json"):
        try:
            data = json.loads(f.read_text())
            if data.get("task_id") == task_id and data.get("status") == "active":
                # check not expired
                return data
        except Exception:
            continue
    return None
```

### 3. allowed_paths 決定 logic

```python
if explicit_allowed_paths_arg:
    allowed_paths = parse_comma(explicit_allowed_paths_arg)
elif (lease := find_active_lease_for_task(task_id)):
    allowed_paths = lease.get("allowed_paths", [])
else:
    print("allowed_paths required: provide --allowed-paths or have an active lease for this task", file=sys.stderr)
    sys.exit(2)

if not allowed_paths:
    print("allowed_paths cannot be empty", file=sys.stderr)
    sys.exit(2)
```

### 4. diff_budget も調整

queue item の `diff_budget` は user-supplied allowed_paths のスケールに合わせる:
