## apply.sh Integration Method

`scripts/authority/apply.sh` should be updated in a follow-up task whose `allowed_paths` includes that file:

1. Detect principal:
   ```bash
   principal="$("$SCRIPT_DIR/principal-detect.sh" ${role_arg:+--role "$role_arg"})"
   ```
2. Run RBAC before OS mutation check:
   ```bash
   rbac_json="$("$SCRIPT_DIR/role-matrix-check.sh" --principal "$principal" --operation "$operation" --target "$target")"
   rbac_allowed="$(jq -r '.allowed' <<<"$rbac_json")"
   ```
3. Reject immediately when `rbac_allowed != true`.
4. Run existing `os-mutation-check.sh`.
5. Proceed only when both RBAC and OS mutation checks return `allowed: true`.
